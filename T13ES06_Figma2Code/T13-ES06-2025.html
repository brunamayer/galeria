GALERIA DE EXERCÍCIOS – Figma to Code

Esta galeria apresenta os componentes de interface desenvolvidos por grupos de estudantes do Inteli. Cada grupo projetou um componente no Figma com Auto Layout e uso do Dev Mode, e posteriormente implementou o componente usando React Native ou HTML/CSS, com base nas medidas, estilos e estrutura de código obtidas do próprio protótipo.

Relato da Atividade
Nesta atividade, foi desenvolvido um componente a partir do protótipo no Figma, utilizando medidas e estilos obtidos via Dev Mode. A implementação foi feita com React Native e exigiu pequenos ajustes manuais para garantir fidelidade ao layout original.
